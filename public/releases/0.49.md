# Version 0.49
This update expands swap capabilities, introduces new authentication options, and improves blockchain coverage across the app.

**1. SWAP IMPROVEMENTS**

We’ve significantly enhanced the Swap experience with more providers, better routing, and improved safety.

What’s new:

New swap providers added:

- Barter
- CCE.Cash
- Pegasus Swap
- Circle Swap

Re-enabled popular DEX providers:

- Uniswap
- PancakeSwap

Additional improvements:

- Upgrade to THORChain Rapid Swaps for faster execution
- Hidden providers with high slippage
- Updated and improved risk scores across providers
- Added the ability to see provider contact details in case of issues

We’re making swaps faster, safer, and more transparent.

**2. NEW SWAP CAPABILITIES**

Added support for Zcash shielded swaps (via Exolix)

**3. WALLET & SECURITY**

Added Passkey support for wallet creation and recovery

This introduces a more modern and convenient way to access your wallet.

**4. NEW BLOCKCHAIN INTEGRATIONS**

- Added Solana integration (iOS)
- Added Zano integration (Android)

**5. NETWORK & INFRASTRUCTURE**

Added PrivateAlps nodes for:

- Monero
- Bitcoin
- Litecoin

Improving reliability and connection stability.

**6. RETURNED FEATURES**

Restored Transaction History

**7. IMPROVEMENTS**

- Minor UI and UX improvements across the app
- Improved performance and stability
- Fixed multiple bugs and edge cases

**8. ZCASH IMPROVEMENTS**

We’ve improved Zcash support to make it more reliable and flexible.

- Updated Zcash to the latest version
- Added ability to select Zcash server
- Added resend functionality for Zcash transactions

**9. PAYMENT IMPROVEMENTS**

Added Open CryptoPay QR scanning support

We’re continuing to improve Unstoppable Wallet to make it more powerful, secure, and easier to use every day.
