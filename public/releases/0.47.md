# 🚀 Version 0.47

This update introduces a major upgrade to the Swap module, enhanced support for privacy-focused assets, and significant reliability improvements throughout the app.

**1. MONERO UPGRADES**

As privacy takes the stage, we’ve been stress testing XMR hard lately. The Monero experience is getting better with each release.

- The build fixes multiple crash cases
- Fixed delayed incoming tx display
- Rescan support with adjustable restore (birthday) height

More coming in ~2 weeks:

- Birthday height edit
- Wallet import UX improvements
- Restore from 25/16 word seeds
- Background sync
- Tx info (incl. to-address + backup)

**2. ZCASH UPGRADE**

Zcash support has been significantly strengthened with protocol-level and privacy enhancements.

What’s new:

- Integrated the latest Zcash SDK used by leading wallets
- Shielded address rotation to reduce address reuse and strengthen privacy
- Improved stability of shielded transactions

**3. UPGRADED PHISHING & ADDRESS POISONING PROTECTION**

We’ve upgraded our built-in spam and address poisoning protection.

This update enhances detection logic to handle more advanced attack patterns, including higher-value transfers designed to bypass basic filters.

The protection engine is now smarter and more adaptive, strengthening transaction safety without affecting normal usage.

If a user attempts to interact with a suspicious address, the wallet warns or blocks the transaction before it is broadcast.

**4. IMPROVEMENTS**

- Improved overall app stability and performance
- Fixed multiple edge-case crashes
- Enhanced reliability across wallet sync and background processes- rescan support with adjustable restore (birthday) height
