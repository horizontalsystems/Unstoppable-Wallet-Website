# Version 0.48

This update brings a major evolution of the Swap experience, expands blockchain support, and improves flexibility for advanced users.

**1. SWAP EXPERIENCE UPGRADE**

We’ve redesigned the Swap module to make it simpler, clearer, and easier to use.

What’s new:

• New Swap tab with a cleaner interface
• Sort providers by best rate, by fastest execution
• Risk scores for providers to better understand potential risks before swapping
• Improved provider visibility to help you choose more confidently

We’re making swaps more transparent - so you can stay in control at every step.

**2. NEW SWAP PROVIDER**

Added support for Exolix

This expands available liquidity and gives users more flexibility when choosing swap routes.

**3. NEW BLOCKCHAIN SUPPORT**

Added support for the Zano network

We continue expanding support for privacy-focused and emerging ecosystems.

**4. TRON IMPROVEMENTS**

• Private key export/import for Tron wallets
• Added Blockchain settings with custom RPC support for Tron

These updates improve flexibility and bring Tron closer to feature parity with other networks.

**5. iOS IMPROVEMENTS**

• Simplified password storage for wallet backups

This makes the backup process easier and more convenient for iOS users.

**6. IMPROVEMENTS**

• Improved overall app performance and responsiveness
• Fixed multiple bugs and edge-case issues
• Enhanced stability across core wallet operations
